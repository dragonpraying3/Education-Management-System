Tuition Centre Management System

This is a Python-based Tuition Centre Management System built to manage courses, students, teachers, staff, and related activities. The system handles student enrolment, course management, grading, attendance, timetables, and reports.

Features
Admin:
-Manage user accounts (Admin, Teacher, Staff, Student).
-Manage courses and class schedules.
-Generate reports for attendance, student performance, and finances.

Student:
-Create account and update personal info.
-Enrol in courses and access course materials.
-View grades and submit feedback.

Teacher:
-Create and manage courses.
-Enrol and manage students.
-Grade assignments and exams; track attendance.
-Generate reports for Admin review.

Staff:
-Register students and manage student records.
-Schedule classes and allocate classroom resources.
-Manage events and communicate with students, parents, and teachers.

Tech Stack
-Python
-File-based storage (text files) for users, courses, and records
-Console-based interface

How to Use
-Run the main Python file.
-Log in with your user role.
-Navigate through the menus to manage students, courses, schedules, grades, or reports.
